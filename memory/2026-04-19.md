### Lessons Learned:
- When designing UI components, create reusable styling systems early
- SVG pattern fills require special handling in `<defs>` section
- Polygon position editing requires translating all points together
- Database migrations should be idempotent (use `IF NOT EXISTS`)
- Button systems should be defined in one place for easy maintenance
